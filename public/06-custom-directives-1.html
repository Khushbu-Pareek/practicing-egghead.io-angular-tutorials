Custom Directives - 1

Using a custom element

The following alert is made using a custom directive using a custom element - <generic-error></generic-error>

Generated Output

<generic-error>
    <div class="alert alert-danger"> Whoops! Something went wrong!   :(</div>
</generic-error>

Using a custom attribute

The following alert is made using a custom directive using a custom attribute - generic-success

The HTML looks like this... <div generic-success></div>

Generated Output

<div generic-success>
    <div class="alert alert-success"> Everything's Alright! :)</div>
</div>
NOTE : What angular really does is whenever it finds a directive in the scope, it puts the provided template inside that element. See the generated output above.

The name & usage of the directives is important. Within javascript, the directive names are camelCased. While using them in html, they should be lowercased and separated by a dash( - )

Example : genericError --> generic-error

module.directive takes the normalized directive name followed by a factory function.
Read more at... https://docs.angularjs.org/guide/directive
